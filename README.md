## Seq2Seq-Framework-with-Mechanisms

A Seq2Seq framework based on pytorch which implements a Seq2Seq model using rnn with below privileges:

* Attention mechanism
  * Encoder-Decoder Attention
  * Inter-Decoder Attention
* Pointer Mechanism
* Coverage Mechanism
* Teacher Forcing
  * Raw
  * Scheduling
* Different RNN Types\
  * RNN
  * LSTM
  * GRU
